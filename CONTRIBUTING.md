# Contributing to CA-Strategy

## About
Strategic planning documents, assessments, and governance frameworks for California government modernization initiatives.

## Content Types
- Policy briefs and position papers
- Technical assessments
- Implementation roadmaps
- Governance frameworks

## Reporting Issues
- Use GitHub Issues for corrections or suggestions
- Include document name and section reference

## Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make changes with clear commit messages
4. Submit PR against `main` branch

## Style Guidelines
- Use plain language (aim for 8th grade reading level)
- Include executive summaries
- Cite sources and provide references
- Follow accessibility guidelines (WCAG 2.1 AA)

## Document Formats
- Markdown for source content
- PDF exports for distribution
- Presentations in reveal.js or PPTX

## Questions?
Open a GitHub Issue.
